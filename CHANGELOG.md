# 更新日志

所有重要的更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/),
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-11

### 新增
- 核心连接池管理功能
  - 支持动态扩容/缩容
  - 连接使用时长监控
  - SQL 执行耗时统计
  - 线程安全的连接池实现
- 数据源配置管理
  - 支持多数据源配置
  - 配置加密存储(AES-256)
  - 环境变量和 JSON5 配置支持
- SQL 执行器
  - SQL 执行超时控制
  - 参数化查询支持
  - 事务管理
  - 批量执行支持
- 安全机制
  - SQL 白名单/黑名单
  - 危险 SQL 拦截
  - 执行权限控制
- 监控功能
  - 连接池状态监控
  - 性能指标收集
  - 日志记录
- 扩展性
  - 自定义钩子函数
  - 多数据库类型预留接口
  - 插件化架构

### 技术栈
- Python 3.8+
- pymysql >= 1.0.0
- DBUtils >= 2.0.0
- python-dotenv >= 0.19.0
- cryptography >= 3.4.0
- psutil >= 5.8.0
- json5 >= 0.9.0
